### Description

### Local Development
To locally test this project run the following commands:
```bash
# Clone this repo
git clone https://github.com/Adespinoza/bitcoin-price-dashboard.git

# cd into it
cd bitcoin-price-dashboard

# Install dependencies
npm install

# Run Development
npm run dev
```
